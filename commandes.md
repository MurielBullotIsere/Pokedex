# apprentissage de node.js et Mongo.db

## création du package.json
```bash
npm init -y
```
## installation de nodemon (package-lock.json)  
/node_modules s'installe quand on utilise la commande install
```bash
npm install nodemon
```
## Installation de express 
```bash
npm install express
```
## Pour avoir les détails  
```bash
npm fund
```
## Installation de Mongo.db
```bash
npm install mongoose
```
## Installation de Cors
```bash
npm install cors
```

## Démarrer le projet
Lancer MangoDB Compass qui se trouve sur le bureau : sélectionner localhost:27017.  
Dans le terminal, lancer la commande
```bash
nodemon server.js. 
```
